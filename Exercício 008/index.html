Principais formatações

Negrito/Destaque

Nessa frase temos um exemplo de texto em negrito usando usando a tag <b> (que não é recomendado pois usa apenas forma)

já nesse temos usando a tag <strong> que além de forma ele foca em semantica significado (com semantica)

Itálico/Ênfase

Nesta frase um termo em itálico usando a tag <i>(que não é recomendado pois usa apenas forma)

Já nesse usamos a tag <em> (com semantica)

Texto marcado

Podemos criar também um texto marcado com a tag <mark>

e aqui temos com a cor padrão

Texto grande e pequeno

Estamos criando um texto grande(desatualizado) usando <big> e um pequeno usando a tag <small> (não desatualizado)

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins>

Existe também a sublinhada usando a tag <u> (ruim)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inseirir coisas do tipo H2O usando a tag <sub>
